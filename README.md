# The Bath Bloodbath - Police Investigation Game

An interactive, immersive police investigation game set in Bath, UK. Players take on the role of an investigator with the Avon & Somerset Police to solve "The Bath Bloodbath" case.

## Features

- **Digital Evidence Locker**: Collect and examine photos, documents, and CCTV footage.
- **Email System**: Communication platform for receiving case updates and forensic reports.
- **Suspect Profiles**: Track and analyze suspects, their roles, and alibis.
- **Interactive Timeline**: Build a chronological view of events as you uncover evidence.
- **Task Management**: Solve puzzles and verify information to progress through the investigation phases.

## Technology Stack

- **HTML5/CSS3**: Custom UI with a modern, dark-themed police terminal aesthetic.
- **JavaScript (Vanilla)**: Core game logic, state management, and interaction.
- **Google Fonts**: Inter and JetBrains Mono for a clean, professional look.

## Local Development

To run the game locally:

1. Clone the repository.
2. Open `index.html` in any modern web browser.

## Deployment

This project is ready for deployment on **Vercel**:

- It's a static site, so no build step is required.
- Simply connect this GitHub repository to Vercel, and it will deploy the `index.html` automatically.
